### 2. Tavily AI Search

让 AI 拥有实时网络搜索能力。专为 AI 应用设计的搜索 API，返回结构化内容而非网页链接。

- [查看详情](./22-tavily-ai-search.md)
- [GitHub](https://github.com/veithly/tavily-search)
- [Python SDK](https://github.com/tavily-ai/tavily-python)
- [官方文档](https://docs.tavily.com)

**主要功能：**

- Web 搜索（结构化结果）
- 内容提取（从 URLs）
- 深度研究模式
- 支持 OpenClaw 集成

---

### 3. ai-news-zh

中文 AI 科技日报自动采集与推送。从 The Verge、Wired、TechCrunch 等英文源抓取最新 AI 资讯，自动翻译整理为中文。

- [查看详情](./29-ai-news-zh.md)
- [GitHub](https://github.com/openclaw/skills)（路径：skills/aizain/ai-news-zh）

**主要功能：**

- RSS 多源聚合
- 自动翻译为中文
- 定时推送（飞书/Telegram/Discord）
- 分类整理

---

→ 选择 **Tavily AI Search**，专为 AI Agent 优化

**需要每日中文 AI 简报？**
→ 选择 **ai-news-zh**，自动翻译聚合英文源

**需要简单资讯整理？**
