# 📈 营销商业

> 该分类下共有 4 个 Skills，涵盖商业分析、营销策略、创业验证和多智能体决策场景

---

## 📚 Skills 列表

### 1. mckinsey-consultant

**麦肯锡顾问** — McKinsey Style 商业问题解决系统

将 McKinsey Problem Solving 方法论系统化为完整工作流，从商业问题分析到专业 PPT 端到端输出。

- [查看详情](./mckinsey-consultant.md)
- [GitHub](https://github.com/fleurytian/awesome-claude-skills)

**适用场景**：商业尽职调查、战略规划、竞争分析、市场进入研究、运营优化

---

### 2. validate-idea

**创业想法验证** — 在投入时间之前验证商业想法的价值

基于《极简创业者》方法论，帮助创业者在开发之前系统性地验证想法是否值得追求。

- [查看详情](./validate-idea.md)
- [GitHub](https://github.com/slavingia/skills)

**适用场景**：创业初期验证、产品方向选择、市场调研辅助、MVP 范围界定

---

### 3. marketing-ideas

**营销方案设计** — 让 AI 为你设计营销策略和创意

来自 40+ 营销技能库，提供内容营销、SEO、增长黑客等各种营销策略的灵感和方案。

- [查看详情](./marketing-ideas.md)
- [GitHub](https://github.com/coreyhaines31/marketingskills)

**适用场景**：SaaS 产品推广、内容营销规划、SEO 优化、增长黑客策略、营销活动策划

---

### 4. roundtable

**多智能体辩论团** — 一个智囊团帮你分析复杂问题

通过 3 个专业子智能体（学者、工程师、创意师）并行思考，交叉辩论后给出综合建议。

- [查看详情](./roundtable.md)
- [GitHub](https://clawhub.ai/robbyczgw-cla/roundtable)

**适用场景**：复杂商业决策、技术方案评估、投资分析、多角度问题分析
