# ⚡ 效率工具

> 提升工作与学习效率的 Skills 分类，涵盖技能优化、知识生成等效率提升工具

## 📚 Skills 列表

本文档目录包含以下 **2 个** 效率工具类 Skills：

---

### 1. darwin-skill

> 自动评测优化，让你的 Skill 进化

**核心功能：** 自动评测 Skills 的表现，发现问题并提供优化建议，实现 Skills 的持续进化。

- ✅ 自动性能评测
- ✅ 问题诊断与定位
- ✅ 优化建议生成

**安装命令：**
```bash
npx skills add alchaincyf/darwin-skill
```

**相关链接：**
- [GitHub](https://github.com/alchaincyf/darwin-skill)

---

### 2. knowledge-site-creator

> 将知识生成学习网站，打造个人知识库

**核心功能：** 将输入的知识内容自动生成结构化的学习网站，便于展示和传播。

- ✅ 知识结构化整理
- ✅ 自动生成网站页面
- ✅ 多格式内容支持

**安装命令：**
```bash
npx skills add joeseesun/qiaomu-knowledge-site-creator
```

**相关链接：**
- [GitHub](https://github.com/joeseesun/qiaomu-knowledge-site-creator)

## 📖 通用安装命令格式

```bash
# 通过 npx skills 安装
npx skills add <GitHub URL> --skill <skill-name>

# 通过 OpenClaw 安装
openclaw skills install <skill-name>

# 通过 ClawHub 安装
clawhub install <author/skill-name>

# 直接克隆到本地
git clone <repo-url> ~/path/to/skill
```
