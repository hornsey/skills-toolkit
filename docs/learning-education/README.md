## 📚 Skills 列表

---

### 1. 女娲

> 蒸馏高手的思维方式，将专家经验转化为可复用技能

**核心功能：** 将领域专家或高手的思维方式和处理问题的模式蒸馏成独立的 Skills。

- ✅ 专家思维模式提取
- ✅ 经验知识化沉淀
- ✅ 跨领域能力迁移

**安装命令：**
```bash
npx skills add alchaincyf/nuwa-skill
```

**相关链接：**
- [GitHub](https://github.com/alchaincyf/nuwa-skill)

---

### 2. cangjie-skill

> 将书本转换为 Skills，让阅读更高效

**核心功能：** 将书籍内容自动转换为结构化的 Skills 格式，便于 AI 学习理解和应用。

- ✅ 书籍内容结构化
- ✅ 知识要点提取
- ✅ 互动式学习体验

**安装命令：**
```bash
npx skills add kangarooking/cangjie-skill
```

**相关链接：**
- [GitHub](https://github.com/kangarooking/cangjie-skill)

## 📖 通用安装命令格式

```bash
# 通过 npx skills 安装
npx skills add <GitHub URL> --skill <skill-name>

# 通过 OpenClaw 安装
openclaw skills install <skill-name>

# 通过 ClawHub 安装
clawhub install <author/skill-name>

# 直接克隆到本地
git clone <repo-url> ~/path/to/skill
```
