# 🌐 浏览器自动化

> 该分类下共有 1 个 Skills，专注于让 AI 代理控制浏览器完成网页自动化任务

## 📚 Skills 列表

### 1. browser-use

让 AI 代理通过自然语言指令控制浏览器的开源库，自动完成网页操作任务

- [查看详情](./browser-use.md)
- [GitHub](https://github.com/browser-use/browser-use)
- [官方文档](https://docs.browser-use.com/)

#### 核心特性

- 🤖 **自然语言驱动**：用日常语言描述任务，无需编写 XPath 或 CSS 选择器
- 🔗 **多 LLM 支持**：兼容 Claude、GPT-4、Gemini 等主流大语言模型
- 🌐 **真实浏览器**：基于 Playwright/CDP 控制真实 Chrome/Firefox 浏览器
- 📋 **多步骤工作流**：自动规划并执行复杂的浏览器操作序列
- 🛡️ **自愈能力**：页面变化时自动调整策略（Browser Harness）
