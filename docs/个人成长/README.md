## 📚 Skills 列表

本文档目录包含以下 **2 个** 个人成长类 Skills：

---

### 1. self-improving-agent

> 让你的 AI 每日持续进化，实现自我提升

**核心功能：** 让你的 AI 具备每日持续学习和进化的能力，通过定期反思和优化提升表现。

- ✅ 每日自我复盘与改进
- ✅ 持续学习与知识积累
- ✅ 目标追踪与进度评估

**安装命令：**
```bash
npx skills add https://github.com/charon-fan/agent-playbook --skill self-improving-agent
```

**相关链接：**
- [GitHub](https://github.com/charon-fan/agent-playbook)

### 2. PUA Skill

> 让 AI 迸发活力提升能动性，激发积极性和创造力

**核心功能：** 通过特定提示词技巧激发 AI 的活力和能动性，使其更加积极主动。

- ✅ 提升 AI 响应质量
- ✅ 增强 AI 主动性
- ✅ 激发创造力与活力

**安装命令：**
```bash
npx skills add https://github.com/tanweai/pua --skill pua
```

**相关链接：**
- [GitHub](https://github.com/tanweai/pua)

## 📖 通用安装命令格式

不同 Skill 的安装方式可能不同，常见格式：

```bash
# 通过 npx skills 安装
npx skills add <GitHub URL> --skill <skill-name>

# 通过 OpenClaw 安装
openclaw skills install <skill-name>

# 通过 ClawHub 安装
clawhub install <author/skill-name>

# 直接克隆到本地
git clone <repo-url> ~/path/to/skill
```
